classList

这个元素返回的是dom元素的class 列表,这个属性本身是只读的，但是我们可以通过几个方法去操作元素的class

  • 可用方法：

    • add ： 添加一个 class 到元素的 class 列表中
    • remove ： 从元素的 class 列表中移除一个 class
    • toggle ： 切换一个 class 是否存在于一个元素的 class 列表中
    • contains ： 查询一个指定的 class 是否存在于元素的 class 列表中
